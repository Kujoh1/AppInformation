# AppInformation

Legal and information pages for apps, served via GitHub Pages.

## Structure

Each app has its own folder with `privacy.html`, `terms.html`, and optionally `index.html`:

- **BabyZoo/** – BabyZoo app
  - [Privacy Policy](BabyZoo/privacy.html)
  - [Terms of Use](BabyZoo/terms.html)

## URLs (GitHub Pages)

After enabling **GitHub Pages** for this repo (Settings → Pages → Source: main branch):

- BabyZoo: `https://<username>.github.io/AppInformation/BabyZoo/`
- Privacy: `https://<username>.github.io/AppInformation/BabyZoo/privacy.html`
- Terms: `https://<username>.github.io/AppInformation/BabyZoo/terms.html`

Replace `<username>` with your GitHub username (e.g. `kujoh1` → `https://kujoh1.github.io/AppInformation/BabyZoo/privacy.html`).

## Subdomains

You can point your own subdomains (e.g. `babyzoo.yourdomain.com`) to these paths via DNS (CNAME to `<username>.github.io`) and GitHub Pages custom domain, or by redirecting to the URLs above.
